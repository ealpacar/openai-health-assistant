# openai-health-assistant
Demo project for AI Applications in Medicine field class.
Tıpta Yapay Zeka Uygulamaları final ödevi için yaptığım demo proje.

# EN
- It needs the libraries first , pip install -r gereklilik.txt to install.
- Needs a working OpenAI API Key.
- Rate limits problems still do exist in free-plan
- app.py file is added later to create an UI.
Its, demo, made with some (especially ui/dl part) help from Claude AI.

# TR
- Öncelikle kütüphaneler kurulmalı, pip install -r gereklilik.txt kurmak için komut olarak kullanılabilir.
- Çalışan bir OpenAI API Key'i gerektirir.
- Ücretsiz plan için Rate limit sorunu vardır.
- app.py dosyası sonradan koda ui eklemek için eklenmiştir
Demo'dur ve özellikle (DL/UI) kısımları için Claude AI'ın yardımı alınmıştır.
